# 🔴⚫ Análise de Desempenho do Flamengo – Temporada 2025

Projeto de análise de desempenho do Flamengo no Campeonato Brasileiro, desenvolvido inicialmente no **Excel** e posteriormente evoluído para um **dashboard web em Python com Streamlit**.

---

## 🚀 Dashboard Online (Streamlit)

👉 https://flamengo-performance2025-dashboard-4yrygqr475f6pbb6zfqncm.streamlit.app

O dashboard apresenta KPIs de desempenho, análise de gols e a relação entre resultado do intervalo e resultado final.

---

## 1️⃣ Coleta de Dados e Análise no Excel
📁 Pasta: `01_coleta_e_excel/`

- Coleta de dados de partidas via API Football-Data
- Tratamento e modelagem dos dados
- Dashboard analítico desenvolvido diretamente no Excel
- KPIs, gols feitos x sofridos e análise de viradas

## 2️⃣ Dashboard Web com Python e Streamlit
📁 Pasta: `02_dashboard_streamlit/`

- Consumo de dados via API
- Engenharia de dados com Pandas e NumPy
- Visualizações com Matplotlib
- Dashboard interativo publicado via Streamlit Community Cloud
